🔧 TROUBLESHOOTING
------------------

❌ "Three.js not loaded" Error:
   - Check internet connection (CDN links)
   - Open DevTools → Network tab
   - Verify Three.js and GSAP loaded

❌ No 3D Effects Visible:
   - Check console for "Running in CSS fallback mode"
   - GPU may not support WebGL
   - Settings panel → Enable 3D Effects

❌ Performance Issues:
   - GPU tier detection should auto-adjust
   - High tier = 500 particles
   - Medium tier = 200 particles
   - Low tier = CSS fallback
   - Manually toggle particles off in settings

❌ Animations Not Working:
   - Check "prefers-reduced-motion" setting
   - Use settings panel to force enable
   - Verify GSAP loaded in Network tab
